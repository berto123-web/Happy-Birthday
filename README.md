# The Time Capsule Gallery

A high-fidelity birthday website mockup built with React + Vite.

## Run locally

```bash
npm install
npm run dev
```

## Personalization

- Recipient name is set in `src/components/TimeCapsulePage.jsx` and passed to `HeroSection`.
- Countdown target date is configured in `src/components/HeroSection.jsx` via `targetMonth` and `targetDay`.
- Timeline memories are configured in the `timelineItems` array in `src/components/TimeCapsulePage.jsx`.

## Notes

- All imagery is intentionally stylized placeholder content.
- Visual direction follows a warm nostalgic palette: sepia, deep cream, and muted gold.
